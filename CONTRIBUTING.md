# CONTRIBUTING.md

## Como rodar localmente
```bash
cd PETEQFURG
python -m http.server 8080
# abra http://localhost:8080/
```

## Branches
- `main` = produção
- `feature/*` = novas melhorias
- `fix/*` = correções

## Pull Requests
- Abra PR para `main`
- Solicite revisão dos Code Owners
- Siga o checklist do template
